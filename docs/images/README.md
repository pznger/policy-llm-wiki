# 展示用截图目录

将 **LLM-Wiki 查询与可溯源答题** 的演示图放在本目录，并在根目录 [`README.md`](../../README.md) 的「效果展示」一节引用。

## 建议准备的图片

| 文件名（建议） | 内容说明 |
| --- | --- |
| `wiki-tree.png` | `wiki/<规范名>/` 目录树：章节目录、`-index.md`、正文 `.md` |
| `index-toc.png` | 某章 `-index.md` 的「条目录」与「条文摘要」 |
| `query-flow.png` | Agent 先读索引再读正文的查询路径示意（可用导图） |
| `traceability-table.png` | qa-test 答案中「关键溯源条文」表格 + 「答案」对照 |
| `pipeline-overview.png` | 可选：PDF → wiki 全流程一张总览图 |

## 在 README 中引用

```markdown
![Wiki 目录结构](./docs/images/wiki-tree.png)
```

图片宽度建议 1200～1600px，PNG 或 WebP，便于 GitHub 深色/浅色主题阅读。
